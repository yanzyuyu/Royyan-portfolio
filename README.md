### 1. PerfMax — Android Performance Optimization Module
**Teknologi:** Bash Scripting

Universal PerfMax adalah modul Android berbasis root yang menerapkan optimasi sistem dan kernel low-level di berbagai chipset. Modul ini secara dinamis menyesuaikan perilaku CPU, GPU, memori, scheduler, dan I/O untuk memprioritaskan performa maksimum.

[Download .zip](https://github.com/yanzyuyu/PerfMax/archive/refs/heads/main.zip)
